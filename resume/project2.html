Project Description

This project extends a previously developed PL/0 compiler to support additional language features including procedure declarations, call statements, and if-else-fi control structures. The compiler performs recursive descent parsing with error detection, manages a symbol table across lexical levels, and generates intermediate assembly code for execution on a virtual machine.

The virtual machine was also enhanced to support the modulo operation (OPR 0 11). If no errors are detected, the compiler outputs a readable assembly listing, displays the symbol table, and generates a numeric instruction file for execution.

Key Features

  • Recursive descent parsing with immediate lexical, syntactic, and semantic error detection
  • Symbol table management across lexical levels using a marking algorithm
  • Support for procedures, call statements, and if-else control structures
  • Intermediate assembly code generation and virtual machine execution
  • Enhanced VM support for modulo arithmetic operations

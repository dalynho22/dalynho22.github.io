Project Description

Developed a two-tier client-server SQL application using Java and MySQL. The system allows users to securely execute SQL DDL and DML commands through a Java GUI client while enforcing authentication and permission-based access. A secondary accountant interface logs database operations for auditing and transaction tracking purposes.

Key Contributions

  • Developed a Java GUI client using JDBC to securely execute SQL DDL and DML queries
  • Implemented user authentication using .properties files with role-based database permissions
  • Built a secondary accountant interface for logging database operations
  • Logged queries across multiple users using a dedicated MySQL operationslog database

Skills Learned

  • Java JDBC database connectivity
  • MySQL database design and management
  • Client-server application architecture
  • User authentication and access control
  • Secure database transaction logging

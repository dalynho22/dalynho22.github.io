Project Description

This project implements a three-tier distributed web-based application using JSP, Servlets, JDBC, and MySQL deployed on Apache Tomcat. The system supports four role-based user interfaces (root, client, data-entry, accountant) with controlled database access.

The middle tier processes SQL commands, executes prepared statements, calls stored procedures, and applies server-side business logic without using database triggers. Database connections are managed through role-specific .properties files.

Key Features

  • Four role-based JSP front-end interfaces with secure authentication
  • Servlet-based middle tier handling SQL queries and updates
  • JDBC integration using Statement, PreparedStatement, and CallableStatement
  • Server-side business logic for supplier status updates
  • Remote stored procedure execution for accountant-level reporting
